# LoveTI

LoveTI is a React + Vite relationship tendency quiz project.

## Deployment

The repository includes GitHub Pages deployment at `.github/workflows/deploy.yml`.
After pushing to a public GitHub repository and enabling Pages, the site can be served publicly from:

`https://<github-username>.github.io/<repo-name>/`
